# DigiFunzi Auth Login UI

A responsive login page built with **React** and **Tailwind CSS** for the DigiFunzi authentication flow.

## Features

- Responsive, centered login card
- DigiFunzi-inspired branding and colors
- Email and password inputs
- Password visibility toggle
- Accessible labels and button states

## Tech Stack

- React (Vite)
- Tailwind CSS
- PostCSS + Autoprefixer

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Run development server

```bash
npm run dev
```

### 3. Build for production

```bash
npm run build
```

### 4. Preview production build

```bash
npm run preview
```
